# Web Crawler

After unzipping, make the Gradle wrapper executable before running any commands:
```bash
chmod +x gradlew
```

Then build and run:
```bash
./gradlew build
./gradlew run
```

## Running locally
```
./gradlew build
```

```
./gradlew run
```

```
./gradlew test
```

```
./gradlew clean
```

## Package
```
./gradlew distZip
```
